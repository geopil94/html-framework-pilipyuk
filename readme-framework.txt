! 'x' обозначает любой символ !


@media - свойства под определённую ширину экрана устройства.

.container / .container-fluid - внешняя / внешняя flex-оболочка.

.row, .col, .col-xxxxxx - описание гибких строк и столбцов. Их ширины и поведения. Общая ширина страницы - 12 единиц. 
Поэтому все колонки имеют коэффициент ширины от 1 до 12.

-md / -md-xx - medium, группа селекторов используемых для экрана, с шириной не менее 768px.
 
-lg / lg-xx - large, группа селекторов используемых для экрана, с шириной не менее 992px.

-xl / -xl-xx - extra large, группа селекторов используемых для экрана, с шириной не менее 1200px.

.flex-xxxx - свойство флекс-оболочки для определения порядка расположения дочерних флекс-элементов.

.clearfix - распорка.

.float-xxxx - включение/выключения обтекания блочных элементов.

.justify-content-xxx - свойство флекс оболочки, для определения порядка расположения дочерних флекс-элементов
относительно главной оси.

.align-items-xxx - свойство флекс оболочки, для определения порядка расположения дочерних флекс-элементов
относительно поперечные оси.

.align-self-xxx - свойство индивидуального расположения флекс-элемента.

.margin-xx - задание внешних отступов.

.padding-xx - задание внутренних отступов.

.text-xxxx - горизонтальное выравнивание текста.
